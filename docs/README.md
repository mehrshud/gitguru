## Quick Start
Get started with GitGuru in minutes. Simply clone the repository, install dependencies, and run the application.

## Installation
To install GitGuru, follow these steps:
1. Clone the repository: `git clone https://github.com/mehrshud/GitGuru.git`
2. Install dependencies: `npm install`
3. Run the application: `npm start`

## Usage Examples
### Repository Structure Management
gitguru repo create my-repo
gitguru repo add collaborator my-repo @username
### Change Tracking
gitguru changes track my-repo
gitguru changes list my-repo
### Collaboration Tools
gitguru task create my-repo "Task 1"
gitguru task assign my-repo "Task 1" @username
### Code Review Integration
gitguru review create my-repo "Pull Request 1"
gitguru review assign my-repo "Pull Request 1" @username
